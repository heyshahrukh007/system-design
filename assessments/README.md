# Days 1–5 MCQ Question Bank

100 multi-select practice questions for **Days 1–5** of the system design curriculum. Each question has **two or more correct answers** — matching a test where multiple options can be right.

**Validated against source material.** See [validation-report.md](./validation-report.md).

---

## Test Format

- **Type:** Multi-select MCQ (select all that apply)
- **Options:** 4 per question (A–D)
- **Correct answers:** Minimum 2 per question
- **Each question includes:** answer key + explanation + source link

## How to Practice

1. Read the question and select **all** options you believe are correct.  
2. Check **Answer** and **Explanation** — note why each distractor is wrong.  
3. If you missed an option, re-read the **Source** lesson section.  
4. Track weak topics and revisit the corresponding `docs/day-XX/` files.
